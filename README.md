# Weather App 🌦️

Простое приложение погоды с поиском по городу.

## Функции
- Поиск по городу
- Температура, ощущается, влажность, ветер
- Обработка ошибок
- Сохранение последнего города (localStorage)

## Стек
- React + Vite
- JavaScript
- Fetch API

```bash
npm install
npm run dev
